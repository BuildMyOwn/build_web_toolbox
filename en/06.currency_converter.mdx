# {frontmatter.title}

In this chapter, we will build a currency converter tool.

## Create Page

Create `src/pages/currency-converter.astro`:

```astro
---
import MainLayout from '../layouts/MainLayout.astro';
---

<MainLayout>
  <h1>Currency Converter</h1>
  <form id="converter">
    <input type="number" id="amount" placeholder="Amount" />
    <select id="from">
      <option value="USD">USD</option>
      <option value="CNY">CNY</option>
    </select>
    <select id="to">
      <option value="CNY">CNY</option>
      <option value="USD">USD</option>
    </select>
    <button type="submit">Convert</button>
  </form>
  <div id="result"></div>
</MainLayout>
```

## Add Interactivity

Use client-side script:

```astro
<script is:inline>
document.getElementById('converter').addEventListener('submit', async (e) => {
  e.preventDefault();
  const amount = document.getElementById('amount').value;
  const from = document.getElementById('from').value;
  const to = document.getElementById('to').value;
  
  const response = await fetch(
    `https://api.exchangerate-api.com/v4/latest/${from}`
  );
  const data = await response.json();
  const rate = data.rates[to];
  const result = amount * rate;
  
  document.getElementById('result').textContent = `${amount} ${from} = ${result.toFixed(2)} ${to}`;
});
</script>
```
