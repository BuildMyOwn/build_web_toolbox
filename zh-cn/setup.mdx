# {frontmatter.title}

在本章中，我们将设置 Web Toolbox 项目。

## 创建项目

运行以下命令创建新的 Astro 项目：

```bash
npm create astro@latest web-toolbox
cd web-toolbox
npm install
```

## 项目结构

项目结构如下：

```
web-toolbox/
├── src/
│   ├── pages/
│   ├── components/
│   └── layouts/
├── public/
└── package.json
```

## 启动开发服务器

运行以下命令启动开发服务器：

```bash
npm run dev
```

在浏览器中打开 `http://localhost:4321` 查看应用。
