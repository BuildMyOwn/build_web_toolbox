## 更新布局组件

创建 `src/layouts/Layout.astro`：

```astro
---
import '../app.css';
---

<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>Web Toolbox</title>
</head>
<body>
  <nav>
    <a href="/">首页</a>
    <a href="/weather">天气查询</a>
  </nav>
  <main>
    <slot />
  </main>
</body>
</html>
```
